# Data Extractor Tool (Python)

A professional Python automation tool that extracts structured product data from a website and generates clean, business-ready CSV reports.

## Features

- Automated web scraping with pagination
- Data cleaning and deduplication
- Structured CSV output
- CLI support for flexible execution
- Clean and modular project structure

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- lxml

## How to Run

```bash
python -m src.main --pages 3

## Output

-Raw data saved in: data/raw
-Processed data saved in: data/processed

## Use Cases

-Product data extraction
-Market research
-Lead generation
-Business automation
